# SafeScape Course Assets

This folder contains all media files for the SafeScape Fire Safety Course.

## Folder Structure

```
assets/
├── videos/
│   ├── m1.mp4    ✅ Module 1: Fire Triangle video
│   ├── m2.mp4    ✅ Module 2: Alarm sounds video  
│   ├── m3.mp4    ✅ Module 3: Escape plan video
│   ├── m4.mp4    ⚠️ NEEDED - Module 4: Smoke behavior / Get Low & Go video
│   └── m5.mp4    ⚠️ NEEDED - Module 5: Stop Drop Roll / Prevention video
│
└── images/
    ├── m1.png              Module 1 image
    ├── line.png            Module 2 waveform image
    │
    │   Module 4 Card Images (optional - placeholders show if missing):
    ├── card1_smoke_physics.jpg   Card 1: Smoke stratification infographic
    ├── card2_toxic.jpg           Card 2: Toxicity/inhalation risks
    ├── card3_crawl.jpg           Card 3: Correct crawling posture
    ├── card4_darkness.jpg        Card 4: Movie vs real smoke comparison
    ├── card5_filter.jpg          Card 5: Using shirt as filter
    │
    └── hazard_room.jpg     ⚠️ NEEDED - Module 5 hazard hunt background
```
